# Block-Reveal

A Next.js gallery project featuring GSAP's Hyperactive Block Reveal Effect.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Block Reveal Animations** - Hyperactive reveal effects using GSAP
- **Smooth Transitions** - Fluid animations between gallery items
- **Next.js App Router** - Modern routing and layouts
- **Responsive Design** - Adapts to all screen sizes
